# Rick and Morty App

This is a Next.js application that uses Apollo Client to query the Rick and Morty GraphQL API.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies Used

- Next.js 15
- TypeScript
- Apollo Client
- GraphQL
- Tailwind CSS
- ESLint
